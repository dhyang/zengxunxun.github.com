Formatting

Much like the C prinf

puts "%s %d" % ["En taro",12] 
puts "%d" % 300 #decimal
puts "%x" % 300 #hexadecimal
puts "%o" % 300 #octal
puts "%b" % 300 #binary
puts "%e" % (5/3.0) #scientific
# # flag adds a prefix to binary, octal and hexadecimal formats.
puts "%#x" % 300 #0x
puts "%#o" % 300 #0
puts "%#b" % 300 #0b
