Attribute accessors

The attr_reader creates getter methods. The attr_writer method creates setter methods and instance variables for this setters. The attr_accessor method creates both getter, setter methods and their instance variables.

class Car
    attr_reader :name, :price
    attr_writer :name, :price  
    def to_s
        "#{@name}: #{@price}"
    end
end

class Book
   attr_accessor :title, :pages    
end
b1 = Book.new
b1.title = "Hidden motives"
b1.pages = 255
p "The book #{b1.title} has #{b1.pages} pages"

The to_s method

The to_s method belongs to this class.

class Being
    def to_s
        "hello world"
    end
end

b = Being.new
puts b
Operator overloading
class Being
    attr_accessor :radius

    def initialize r
        @radius = r
    end

    def +(other)
        Being.new (@radius + other.radius)
    end

    def to_s
        "output #{@radius}"
    end
end

a1 = Being.new 1
a2 = Being.new 2
puts a1+a2
Class methods

Class methods are called on a class. They cannot be called on an instance of a class.

class Being 
    def self.info1
       "This is a Wood class info1" 
    end

    class << Being 
        def info2
           "This is a Brick class info2" 
        end
    end
end

def Being.info3
   "This is a Rock class info3" 
end

puts Being.info1
puts Being.info2
puts Being.info3
instance method

Instance methods belong to an instance of an object

class Being 
    attr_accessor :info2
    def info1
        "This is info1 object"
    end
end

a = Being.new
puts a.info1

b = Being.new
b.info2 = "This is info2 object"
puts b.info2

c = Being.new
def c.info3
    "This is info3 object" 
end
puts c.info3
Polymorphism
class Animal
    def make_noise 
        "Some noise"
    end
    def sleep 
        puts "#{self.class.name} is sleeping." 
    end
end
class Dog < Animal
    def make_noise 
        'Woof!'         
    end 
end
class Cat < Animal 

    def make_noise 
        'Meow!' 
    end 
end
a = Animal.new 
d = Dog.new
c = Cat.new
puts a.make_noise
puts d.make_noise
puts c.make_noise
a.sleep
d.sleep
c.sleep
